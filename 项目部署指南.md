## 🚀 部署步骤

### 1. 上传项目到NAS

将整个 `home-library` 文件夹上传到NAS。

### 2. 初始化数据库

```bash
cd backend
python init_db.py
```

### 3. 导入书籍数据

```bash
cd backend/sql
python import_books.py
mysql -h 192.168.1.108 -u chenwei -p < import_books.sql
```

### 4. 启动服务

```bash
docker-compose up -d --build
```

### 5. 访问应用

- **前端**：http://NAS_IP:5101
- **API**：http://NAS_IP:5201/api
